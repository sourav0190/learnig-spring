# Internal Working of Spring Boot (IWS)

This project explores the internal workings and core concepts of the Spring Boot framework.

## Default Spring Boot Version Note
> **Note**: This project's `pom.xml` references Spring Boot version `4.0.1`. Please verify if this version is intended or supported as the latest stable releases are in the 3.x series.

## Project Overview
This repository serves as a learning resource and reference for understanding:
- Spring Boot Auto-configuration
- Dependency Injection
- Spring MVC flow
- Data Access with JPA/Hibernate

## Tech Stack
- **Java**: 21
- **Spring Boot**: 4.0.1 (Experimental/Placeholder)
- **Database**: MySQL and H2
- **ORM**: Hibernate

## Getting Started

### Prerequisites
- Java 21
- Maven

### Running the Application
1. Clone the repository.
2. Navigate to the `IWS` directory.
3. Run the application using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```
